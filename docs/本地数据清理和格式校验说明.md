# 本地数据清理和格式校验说明

本文件说明阶段 10：文件格式提醒，以及删除项目时同步清理本地数据。

## 文件格式提醒

新建任务页现在会检查文件格式和任务类型是否匹配。

比如：

- 当前是音频纪要，但选择了 `mp4` 视频文件，会提示不匹配。
- 当前是视频字幕，但选择了 `mp3` 音频文件，会提示不匹配。

格式不匹配时，不允许创建本地任务。

## 删除项目时同步清理

现在删除项目时，会同步清理：

- 这个项目的本地草稿。
- 这个项目的导出记录。
- 这个项目在工作台里的项目记录。

## 大白话解释

以前删除项目，只是从项目列表里删掉。

现在删除项目，会顺手把它相关的草稿和导出记录也清掉，避免本地浏览器里留下没用的数据。

## 当前边界

- 只清理当前浏览器里的本地数据。
- 不影响已经下载到电脑里的文件。
- 不操作真实服务器。
- 不接入 AI。
