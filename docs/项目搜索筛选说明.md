# 项目搜索筛选说明

本文件说明阶段 7：工作台项目搜索和筛选。

## 当前能力

工作台现在可以：

- 按项目名称搜索。
- 按文件名搜索。
- 按文件格式搜索。
- 按项目类型筛选：全部、音频纪要、视频字幕。
- 按项目状态筛选：全部、处理中、已完成、草稿。
- 一键清空筛选条件。

## 空结果提示

如果筛选后没有结果，页面会提示：

- 没有找到符合条件的项目。
- 可以换关键词。
- 可以清空筛选。

## 大白话解释

项目多了以后，用户不用一个个翻列表。

比如：

- 输入 `mp4`，就能找到视频文件。
- 选择 `音频纪要`，只看录音相关项目。
- 选择 `处理中`，只看还没完成的任务。

## 当前边界

- 搜索和筛选只在当前浏览器的本地项目里进行。
- 不连接真实数据库。
- 不搜索真实文件内容。
- 不搜索音频或视频里面说了什么。
