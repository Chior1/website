# 项目草稿保存说明

本文件说明阶段 8 和阶段 11：每个项目独立保存编辑草稿，并支持本地自动保存。

## 当前能力

音频纪要项目可以保存：

- 逐字稿文字。
- 发言人名称。
- 会议摘要。
- 关键结论。
- 待办事项。

视频字幕项目可以保存：

- 字幕文字。
- 字幕开始时间。
- 字幕结束时间。
- 拆分、合并、删除后的字幕列表。

## 怎么使用

进入项目详情页后，直接编辑纪要或字幕。

页面会在你停止输入后自动保存本地草稿。也可以点击顶部的“保存草稿”，手动保存一次。

下次再打开同一个项目，会自动恢复这个项目自己的草稿。

## 保存在哪里

草稿保存在当前浏览器本地。

大白话：

- 不上传到服务器。
- 不同步到其他电脑。
- 清理浏览器数据后会消失。
- 这适合原型阶段，用来验证编辑流程。

## 当前边界

- 不接入 AI。
- 不保存真实音视频。
- 不保存到真实数据库。
- 不做多人协作。
- 自动保存只保存在当前浏览器，不代表已经上传到云端。
- 自动保存有短暂延迟，不是每输入一个字都立刻保存。

## 后续可以增强

- 草稿版本历史。
- 云端保存。
- 多人协作编辑。
