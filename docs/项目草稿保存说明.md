# 项目草稿保存说明

本文件说明阶段 8：每个项目独立保存编辑草稿。

## 当前能力

音频纪要项目可以保存：

- 逐字稿文字。
- 发言人名称。
- 会议摘要。
- 关键结论。
- 待办事项。

视频字幕项目可以保存：

- 字幕文字。
- 字幕开始时间。
- 字幕结束时间。
- 拆分、合并、删除后的字幕列表。

## 怎么使用

进入项目详情页后，点击顶部的“保存草稿”。

下次再打开同一个项目，会自动恢复这个项目自己的草稿。

## 保存在哪里

草稿保存在当前浏览器本地。

大白话：

- 不上传到服务器。
- 不同步到其他电脑。
- 清理浏览器数据后会消失。
- 这适合原型阶段，用来验证编辑流程。

## 当前边界

- 不接入 AI。
- 不保存真实音视频。
- 不保存到真实数据库。
- 不做多人协作。
- 不做自动保存，当前是手动点击“保存草稿”。

## 后续可以增强

- 自动保存。
- 显示最后保存时间。
- 草稿版本历史。
- 云端保存。
- 多人协作编辑。
